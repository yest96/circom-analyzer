# Механизм обнаружения проблем безопасности в цепях доказательств с нулевым разглашением

Инструмент для автоматического обнаружения недоограниченных переменных в арифметических схемах на языке Circom. Механизм сочетает динамический анализ (фаззинг) с символьной верификацией (SMT), что позволяет выявлять уязвимости, недоступные для статических анализаторов.

## Как это работает

1. **Фаззинг** — генерирует случайные входные данные и собирает статистику по диапазонам переменных-свидетелей
2. **Граф ограничений** — строит граф по R1CS-представлению схемы и кластеризует переменные алгоритмом Лувена
3. **SMT-верификация** — проверяет каждый кластер на наличие недоограниченных переменных с помощью решателя CVC5

## Требования

- Python 3.10+
- circom
- cvc5

## Запуск

```bash
python script.py <circuit_path>
```

Например:

```bash
python script.py circuits/multiplier.circom
```
